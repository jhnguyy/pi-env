# Communication Pattern Guide

Smells, not bans.

| Pattern | Issue | Move |
|---|---|---|
| Empty announcement: “Here’s why this matters”, “It’s worth noting” | Warm-up before the point. | Start with the point. |
| Inflated importance: “The stakes are high”, “This is critical” | Importance without consequence. | Name the cost, risk, mechanism, or effect. |
| Manufactured contrast: “Not X. Y.”, “The real problem is…” | Reveal instead of relation. | State the distinction, cause, or tradeoff. |
| Performative certainty: “clearly”, “obviously”, “make no mistake” | Confidence without evidence. | Give the evidence or cut the signal. |
| False intimacy: “I promise”, “we’ve all been there” | Unearned familiarity. | Cut unless the relationship matters. |
| Template rhythm: repeated shapes, stacked fragments, punchline endings | Repeated cadence. | Combine, vary, or make concrete. |
| False agency: “The data tells us”, “The decision emerged” | Abstractions acting like people. | Name the actor when useful. |

Inanimate subjects are fine when precise: “the process exits,” “the server returns 500,” “the cache evicts the key.”
