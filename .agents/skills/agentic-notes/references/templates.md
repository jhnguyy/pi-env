# Minimal Templates

Use these as starting points, not mandatory schemas.

## Current Reference

```markdown
# Title

One-paragraph summary of the current state.

## Context

## Current Understanding

## Evidence

## Open Questions

## Related
```

## Decision Note

```markdown
# Decision — Short Name

Decision: one sentence.

## Context

## Options Considered

## Rationale

## Consequences

## Revisit When
```

## Handoff

```markdown
# Handoff — Task or Project

## Goal

## Current State

## What Changed

## Important Files / Notes

## Risks / Unknowns

## Next Steps
```

## HTML Sidecar Companion Summary

Put this in the canonical Markdown note when HTML is useful but not the source of truth:

```markdown
## Visual Artifact

Generated sidecar: `path/to/artifact.html`

Purpose: what the artifact helps compare, understand, or decide.

Key takeaways:
- ...
- ...
```
